# 🚀 AI Career OS

> Your complete AI-powered career operating system — built for students, freshers & tech professionals.

**Live Demo →** [https://shirapti-nath.github.io/ai-career-os](https://shirapti-nath.github.io/ai-career-os)

---

## 🌐 Two Platforms, One Ecosystem

### 📊 ResumeAI — Career Toolkit
AI-powered resume, LinkedIn, and career tools.

- ✅ ATS Resume Analyzer (0–100 score with keyword gaps)
- ✅ AI Bullet Rewriter — strong verbs + quantified metrics
- ✅ LinkedIn Profile Optimizer — headline, About, SEO keywords
- ✅ Cover Letter Generator — tone-aware, role-specific
- ✅ 10 ATS Templates (5 Student + 5 Professional)
- ✅ Resume Builder with live preview + PDF export

**🚀 AI Career OS Tools:**
- 🛠️ AI Project Generator — portfolio-ready ideas with resume bullets
- 🎤 Mock Interview System — DSA / System Design / HR with AI scoring
- 🎯 Job Readiness Score Engine — gap analysis for your target role
- 🧠 Daily Career Coach — streaks, goals, weekly learning plan
- 🐙 GitHub Portfolio Analyzer — recruiter-perspective scoring
- 🌐 Career Simulation Engine — 3-year salary trajectory simulation

### 🔭 TechPath OS — Career Explorer
Interactive tech career roadmaps for 12 domains.

- Full Stack · AI/ML · DevOps · Cybersecurity
- Data Analyst · Data Engineer · Mobile Dev
- Game Dev · Cloud · Blockchain · QA · UI/UX
- Salary ranges from entry to leadership level
- Skill trees and learning progression paths

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React 18 + Vite + Tailwind CSS + Framer Motion |
| Backend | Node.js + Express REST API |
| Database | MongoDB + Mongoose |
| AI | Claude API (claude-sonnet-4-6) |
| Auth | JWT Bearer Token |
| Demo | Pure HTML/CSS/JS (CDN Tailwind) |
| Deploy | GitHub Pages |

## 🚀 Run Locally

```bash
# Demo (no setup needed)
open index.html

# Or serve with Python
python3 -m http.server 3000

# Full project — Backend
cd backend && npm install && npm run dev

# Full project — Frontend
cd frontend && npm install && npm run dev
```
